# aoc-ts-template

## Installation

npm
```bash
npx degit sebous/aoc-ts-template <name>
```
pnpm
```bash
pnpm dlx degit sebous/aoc-ts-template <name>
```

## How to use
- create input file in `/inputs` folder
- create day solution file in `/src/solutions`
- run day via
```bash
npm start <day>
```
- also see `/src/solutions/01.ts`
